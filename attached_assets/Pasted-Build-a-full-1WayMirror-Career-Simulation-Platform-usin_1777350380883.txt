Build a full 1WayMirror Career Simulation Platform using immersive 3D/VR-style missions.

This should not be a quiz platform. Do not use a fixed Question 1, Question 2, Question 3 structure. These should feel like interactive career missions where students solve realistic problems through choices, tools, sliders, drag-and-drop, voice, object interaction, and AI-generated events.

Branding:

Platform name: 1WayMirror
Dark navy background
Gold/orange accents
Clean white text
Modern, immersive, high-school-friendly feel
CTA: Start Mission

Core requirements:

Every career topic must have a 3D/VR-style interactive environment
Use WebGL/Three.js or similar browser-based 3D
Add “Enter VR” where supported, but also make everything work on laptops
AI generates never-ending missions
Progress is tracked across all categories
Track XP, level, streak, mission history, skill strengths, weak areas, badges, and completion rate
Missions must adapt based on performance
Keep all content fun, realistic, high school appropriate, and challenging but not overwhelming
Avoid PhD-level technical depth
After each mission, automatically generate a new one

Mission engine should include:

Career environment
Real-world problem
Interactive tools
AI-generated obstacles
Budget/time/safety/customer constraints
Multiple possible solutions
Performance score
Feedback based on choices
Suggested next mission
Skill map update
