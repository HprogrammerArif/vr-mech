Dario has SEVEN features, gated by tier per Section 3 table:

1. CHAT WITH DARIO
   - Conversational AI for any career questions

2. COMPARE CAREERS

3. CAREER ROADMAP
   - Personalized step-by-step career plan
   - Includes milestones, education path, skills to develop, timeline
   - Available: Explorer, Builder, Accelerator

4. PERSONALITY INSIGHTS
   - Strengths, learning style, work style analysis
   - Based on platform activity + direct assessment
   - Available: Explorer, Builder, Accelerator

5. CAREER REPORT
   - Comprehensive PDF compiling all student data and analysis
   - Shareable with parents, counselors, mentors
   - Available: Builder, Accelerator

6. OPPORTUNITIES
   - Matched real-world opportunities (internships, programs, 
     scholarships, shadowing, summer programs)
   - Updates regularly based on student profile
   - Available: Builder, Accelerator

7. ACTION ITEMS
   - Specific deadline-driven tasks with progress tracking
   - AI generates next-step recommendations as items complete
   - Available: Accelerator only

- 25% credits remaining: gentle in-app notification
- 10% credits remaining: prominent warning + upgrade prompt
- 0 credits: lock Dario chat, show refresh date and upgrade CTA

SECTION 8: LIVE SPEAKER SESSION GATING
================================================================

Caps per tier:
- Starter: 2 sessions per month, system-assigned random rotation, 
  attendance only, no recordings access
- Explorer: 8 sessions per month, user's choice from full schedule
- Builder: Unlimited live sessions
- Accelerator: Unlimited live sessions

Recording access:
- Starter: No recordings access (locked)
- Explorer/Builder/Accelerator: Unlimited recordings library access

Database tracking:
live_session_attendance table:
- user_id (FK)
- session_id (FK)
- attended_at (timestamp)
- billing_period (string YYYY-MM format)
- is_recording (boolean)

Cap counting logic:
- Count attended_at events in current billing_period where 
  is_recording = false
- Block new live attendance when count >= tier limit
- Recordings (is_recording = true) never count against cap

- Accelerator Annual ($1,999) — 7-day trial

WEBHOOK HANDLERS REQUIRED:

subscription.created:
- Activate user's tier in subscriptions table
- Create dario_credits row with monthly_allocation per tier
- Unlock features per tier
- Send welcome email with tier-specific onboarding

subscription.updated:
- Detect tier change (upgrade or downgrade)
- Upgrade: prorate new credits immediately, unlock features
- Downgrade: schedule change for next billing cycle, retain current 
  credits until period end
- Update subscriptions table

subscription.deleted (cancellation):
- Mark cancel_at_period_end = true
- Access continues until current_period_end
- After period ends, lock features and remove credit allocation
- Preserve user data for 90 days, then anonymize

invoice.payment_failed:
- Mark subscription status = past_due
- Begin grace period (3 retries over 7 days)
- Send notification email + SMS
- After grace period, downgrade to no-access state

customer.subscription.trial_will_end:
- Send conversion reminder 3 days before trial end
- Send final reminder 1 day before
- Auto-convert to paid unless canceled

SECTION 11: UPGRADE/DOWNGRADE FLOWS
================================================================

UPGRADE FLOW:
- One-click via Stripe with prorated billing
- New credits added immediately (prorated to days remaining in period)
- New features unlocked immediately
- New content accessible immediately
- Confirmation email + in-app notification
- Show "What's New" tour highlighting newly unlocked features

DOWNGRADE FLOW:
- Takes effect at next billing cycle (no immediate access removal)
- Existing credits remain available until period ends
- Confirmation: "Your plan will change to [Tier] on [date]. You'll keep 
  your current features until then."
- User can reverse downgrade anytime before period ends

CANCELLATION FLOW:
- "Are you sure?" modal with retention offer (e.g., 50% off next month)
- Access continues until billing period ends
- Confirmation email with cancellation date and reactivation instructions
- Data preserved for 90 days
- After 90 days, account anonymized
